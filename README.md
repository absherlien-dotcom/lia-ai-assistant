# LIA AI — Personal Intelligence

ليا هي مساعد ذكاء اصطناعي شخصي خاص بهشام فقط.

## النسخة الحالية

- واجهة هوية كاملة
- شاشة محادثة فاخرة
- اتصال مع Gemini عبر Railway
- API: `/chat`
- صفحة رئيسية تعمل من نفس رابط Railway

## التشغيل

```bash
npm install
npm start
```

## المتغيرات المطلوبة في Railway

```txt
GEMINI_API_KEY=ضع_مفتاح_جيميني_هنا
```
